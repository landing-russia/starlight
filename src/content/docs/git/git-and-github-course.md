## Инициализация GIT репозитория

```sh
git init
```

```sh
git status
```

## Рабочий процесс GIT

```sh
git add .  # Добавить всё
```

```sh
git commit -m "your commit"
```

```sh
git log
```

```sh
git checkout  # Перейти на ветку или коммит
```
